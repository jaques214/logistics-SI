◆ readString()

bool readString ( char *const  value,
const unsigned int  size,
char const *const  message 
)

Lê uma cadeia de caracteres (string) através da stream stdin.

Parâmetros
value- Apontador para a frase lida.
size- Tamanho máximo da frase a ser lida.
Aviso
: O tamanho da frase deve ter em conta o espaço para o carácter de término da string ('\0')
Parâmetros
messageA mensagem a mostrar ao utilizador
Retorna
1 em caso de leitura com sucesso, 0 caso ocorra um erro.

Exemplo:

char nome[100 + 1];
printf("Introduza o seu nome: ");
if (lerFrase (nome, 100 + 1) == 1)
printf("O nome lido e: %s", nome);
